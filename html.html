What is HTML?

HTML (HyperText Markup Language) is the standard language used to create and design websites. It structures the content on the web. HTML is the foundation of most web pages, as it tells the browser how to display the text, images, and other content on a page.

HTML Basic Structure

Every HTML document follows a basic structure:

<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Your Title</title>
</head>
<body>
  <h1>Hello, World!</h1>
  <p>This is a simple HTML page.</p>
</body>
</html>

This is the basic HTML structure. It starts with the `` declaration to specify the document type, followed by the opening and closing `` tags. Inside the ``, we define metadata like the character set and title, and inside the ``, we define the content that will be displayed on the web page.
